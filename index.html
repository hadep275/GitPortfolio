Project 2: Every child plays

Our solution provides a way for parents to find inclusive playgrounds in Calgary, ensuring every child has access to safe and enjoyable play spaces. Parents can use our web or mobile app to discover the perfect playground for their kids.

For this project, we utilized a range of technologies to create an intuitive and visually appealing platform. We harnessed the power of React.js and CSS to build our website, while incorporating dynamic maps and leveraging the Material-UI (MUI) library for an enhanced user interface.

On the backend, we employed Express and MongoDB to handle data storage, providing functionality for playgrounds, users, reviews, and feedback messages. This seamless integration between React in the frontend and Express/MongoDB in the backend ensures a smooth user experience.
